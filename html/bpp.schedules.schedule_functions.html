bpp.schedules.schedule_functions.create_nighttime_oa_shutoff_sch(PrimaryAirSystem)#

Create a nighttime outdoor air (OA) shutoff schedule for applicable Air Handling Units (AHUs).

Parameters:
PrimaryAirSystemobject

The PrimaryAirSystem object for which the schedule is being generated.

Returns:
YearSchedule

The generated year schedule for the PrimaryAirSystem.

Notes

This function creates a schedule that shuts off outdoor air during unoccupied hours. The start and end hours of occupancy are retrieved from the PrimaryAirSystem’s project attribute. The function creates separate day schedules for occupied and unoccupied days, and assigns them to a week schedule. The week schedule is then assigned to a year schedule, which is returned.

bpp.schedules.schedule_functions.generate_airflow_schedules(SpaceType, schType='Fraction')#

Generate airflow schedules based on the SpaceType and setback_schedule_source.

Parameters:
SpaceTypeobject

The SpaceType object representing the type of space.

schTypestr, optional

The type of schedule to generate. Defaults to ‘Fraction’.

Returns:
airflow_scheduleobject

The generated airflow schedule.

Notes

This function generates airflow schedules based on the SpaceType and setback_schedule_source. If the setback_schedule_source is “Building Occupancy Hours”, it calls the generate_building_occupancy_hours_airflow_schedules function. If the setback_schedule_source is “90.1 User’s Manual”, it calls the generate_ashrae_default_airflow_schedules function. If the setback_schedule_source is not recognized, it prints an error message and calls the generate_ashrae_default_airflow_schedules function as a fallback.

bpp.schedules.schedule_functions.generate_building_occupancy_hours_temperature_setpoint_schedules(SpaceType)#

Generate temperature setpoint schedules based on building occupancy hours.

Parameters:
SpaceTypeobject

The SpaceType object.

Returns:
tuple

The generated heating and cooling schedules.

Notes

This function generates temperature setpoint schedules based on the occupancy hours entered by the user for a given SpaceType. It creates separate day schedules for occupied and unoccupied periods, assigns them to a week schedule, and then assigns the week schedule to a year schedule. The function returns a tuple of the heating and cooling schedules.
