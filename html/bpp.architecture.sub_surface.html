class bpp.architecture.sub_surface.SubSurface(id='551ed92f-6452-4133-b2f7-a24e7992d02a')#

Bases: object

A class to represent a sub-surface in a building.

A sub-surface is a part of a surface that has a different construction or type. Examples of sub-surfaces include windows, doors, and skylights.

Parameters:
idstr, optional

A unique identifier for the sub-surface. If not provided, a random UUID is generated.

Attributes:
namestr

The name of the sub-surface.

uuidstr

The unique identifier of the sub-surface.

The type of the sub-surface. Can be ‘Window’, ‘Skylight’, or ‘Door’.

The name of the construction of the sub-surface.

The construction of the sub-surface.

operablebool

Whether the sub-surface is operable.
