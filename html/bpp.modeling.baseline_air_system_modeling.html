bpp.modeling.baseline_air_system_modeling.assign_baseline_zones_to_air_systems(baseline_json_data)

Edit the zone json data to include system assignments.

Make a dictionary with each space type’s baseline category (for labs and vivaria). Make a dictionary referencing zones to their space types. Create a dictionary with each relevant zone_group_name as … Run through baseline systems to assign zones. Add remaining zone lists associated with building stories to master list. Run through all zone lists to assign each zone to its baseline system.

bpp.modeling.baseline_air_system_modeling.write_baseline_system_to_json(baseline_json_data, system_uuid, cooling_loop_uuid=None, heating_loop_uuid=None)

Construct the json data for a primary air system based on the baseline system data.

Read baseline system from baseline data. Write air system to baseline json with the same uuid as the baseline system. Use VAV system types and 90.1 fan curve if System 5-8. Set economizer. Set energy recovery type and effectiveness. Add baseline fan powers. Add airflow information. Define heating coil and humidification based on baseline system heating type. Define cooling coil based on baseline system cooling type.
