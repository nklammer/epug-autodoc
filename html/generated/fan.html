Object to represent a fan.

Parameters
air_systemcustom type

The custom air system object name that the fan is assigned to.

idstr, optional

The UUID of the object.

fan_positionstr, optional

One of [‘Supply’, ‘Return’, …]

Notes

The following equation describes how part load fan power is determined from fan flow fraction FF and fan curve coefficients. Source

\[PLF = C_1 + C_2(FF^1) + C_3(FF^2) + C_4(FF^3)\]
