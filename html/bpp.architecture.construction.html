class bpp.architecture.construction.Construction(id='0edf5324-d4e6-497e-9727-b2bcb224d3c3')#

Bases: object

Class for construction objects in OpenStudio.

This class represents construction objects in OpenStudio. It can handle different types of constructions: ‘Layered’, ‘F-Factor’, and ‘C-Factor’.

Attributes:
namestr

The name of the construction.

uuidstr

The unique identifier of the construction.

typestr

The type of the construction. It can be ‘Layered’, ‘F-Factor’, or ‘C-Factor’.

material_layerslist

A list of Material objects listed from outside to inside.

material_layer_nameslist

A list of names of the material layers.

material_thicknesseslist

A list of thicknesses of the material layers in inches.

f_factorfloat

The F Factor of the construction. This is used to define constructions that do not have material layers defined.

areafloat

The area of the construction in square feet. This needs to be updated to pull from the model.

perimeterfloat

The perimeter of the construction in feet. This needs to be updated to pull from the model.

c_factorfloat

The C Factor of the construction. This is used to define constructions that do not have material layers defined.

heightfloat

The height of the construction in feet. This needs to be updated to pull from the model.

Write the Construction object to an OpenStudio model.

This method takes an OpenStudio model and writes the Construction object to it. The method handles different types of constructions: ‘Layered’, ‘F-Factor’, and ‘C-Factor’.
