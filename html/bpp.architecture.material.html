bpp.architecture.material module#

class bpp.architecture.material.Material(id='ce772388-073c-46cf-b681-d1453a0aabfb')#

Bases: object

A class used to represent a Material.

A Material object represents a material with its properties such as name, uuid, roughness, thickness, conductivity, density, specific heat, thermal absorptance, solar absorptance, visible absorptance, etc.

write_to_os(model, thickness=None)#

Write the material properties to an OpenStudio model.

This method writes the properties of the material to an OpenStudio model. If the material is an air layer, a no mass material, a simple glazing, or a standard glazing, the corresponding OpenStudio model is created. If none of these conditions are met, a standard opaque material is created.

Parameters:
modelopenstudio.model.Model

The OpenStudio model to which the material properties will be written.

thicknessfloat, optional

The thickness of the material. If not provided, the thickness attribute of the material will be used.

Returns:
openstudio.model.Material

The material written to the OpenStudio model.
